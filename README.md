#MyGoTGBot public version

目前具有的功能：  
1.interactive function  
2.database  
3.login api  
4.webspider  
5.docker /docker compose  
6.nlsc weather spider  

使用時須自行新增設定檔─檔名為config.json且須含以下項目  
{  
    "BotToken": "YourBotToken",  
    "OwnerID": 1366159494,  
    "GroupID": YourGroupID(int),  
    "WeatherToken": "YourNLSCToken",  
    "movieMaxPage": 10
}
